# Torst
A Web Control Panel 浏览器远程关机重启-云盘与在线预览

![image](https://github.com/PJY1548/Torst/blob/main/preview.png)
## -------- 亮点 --------
### 1.广泛兼容各版本系统（Windows7Vxkex可用）
### 2.功能多样（关机重启，云盘，支持预览视频，图片，文档，小说（epub，pdf））
### 3.自定义密码保护
### 4.云盘自定义位置，管理便捷
### 5.适配移动设备，适配深色模式

## -------- 部署方式 --------
### 这里是一般性部署
### 2.下载发布版本，存放至安装目录，右键setup.ps1，使用powershell运行
### 3.正常完成安装程序
### 4.使用run.bat运行或直接使用node server.js
## -------- 关于 --------
### 自启动：可以自行在run.bat中添加定位到你的安装目录，使用任务计划程序实现开机自启动
### 远程使用：
#### --使用cloudflare Tunnel实现零成本云盘（当然不算电费）
#### --使用FRP内网穿透工具
